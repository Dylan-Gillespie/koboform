The KoboOpenRosa project within this workspace is the adapted OpenRosa source 
code, i.e. this is the source tree developed in house building off of the 
OpenRosa FormDesigner.

FormDesigner is the original FormDesigner supporting XForms and written
by Ugandan Daniel.

Jetty Server is, a Jetty Server. This is embedded in the application launcher.

SharedLib is from Daniel's original work on the FormDesigner supporting
XForms as well.
